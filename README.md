#CRUD Entities — Fullstack App

Prosty projekt CRUD z frontendem i backendem w Node.js + Express, korzystający z PostgreSQL.  
Frontend i backend są hostowane razem w jednej usłudze na Render.  

---

##Funkcjonalności

-Dodawanie encji (`name`, `type`, `description`, `extra_text`)  
-Edytowanie encji  
-Usuwanie encji  
-Lista encji aktualizuje się automatycznie po każdej zmianie  
-Frontend: HTML + JS, backend: Node.js/Express + PostgreSQL
